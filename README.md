### 前端 Side Project

* **名字筆畫工具**

  [Demo](https://shinenic.github.io/strokes-operation-react/)  

  [code & features](https://github.com/shinenic/strokes-operation-react)

  前端提供介面讓使用者計算中文漢字筆劃、名字總筆劃組合並記錄之網頁應用程式

  * React, Redux 建置前端頁面

  * 實作 Pagination 完成長列表優化

  


* 卓著歌曲快速搜尋

    * React
    * 長列表效能優化
